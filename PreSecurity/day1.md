What is Cyber Security?
Cyber security is the practice of protecting systems, networks, and data from digital attacks.

Offensive Security
•	Simulates hacker behavior to find vulnerabilities 
•	Also called ethical hacking / penetration testing 

Defensive Security
•	Focuses on protecting systems and responding to attacks 
•	Includes: 
o	Detecting attacks 
o	Investigating incidents 
o	Securing compromised systems 
Example:
•	Stopping an attack in progress = containment 

Terminal
•	Interface used to interact with a system using commands 
•	Used in cybersecurity to run tools and scripts 

Linux
•	Common operating system used in cybersecurity 
•	Often used in virtual machines (VMs) 
•	Preferred for security tools and testing 

Key Cyber Security Roles
Ethical Hacker / Pentester
•	Tests systems like an attacker 
•	Finds vulnerabilities in websites and applications 

Network Engineer
•	Builds and fixes network systems 
•	Ensures secure communication between devices 

DevSecOps Engineer
•	Integrates security into software development 
•	Prevents vulnerabilities during the building process
